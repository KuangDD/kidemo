
## kidemo
kid demo,像小孩学习一样的demo。

### 异步编程
async_demo

* 异步函数的写法。
* 异步函数怎么执行。
* 异步函数怎么获取返回的结果。

### Tornado异步访问
tornado_httprequest_demo

* Tornado怎么访问web接口。
* Tornado怎么获取接口返回的数据。
* 怎样获取到异步返回的数据。
* 用callback的方式处理Tornado返回的数据。
* 用常规方式处理Tornado返回的数据。

### Tornado服务部署
tornado_httpserver_demo

* 怎样部署Tornado的web服务或接口。
* GET和POST方法的构建方法。
